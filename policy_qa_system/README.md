# 政策问答系统

基于 RAG 技术的智能政策咨询系统

## 系统架构

- **后端框架**: Flask
- **数据库**: SQLite
- **向量数据库**: ChromaDB
- **大语言模型**: 智谱 GLM-4.7-Flash
- **嵌入模型**: 腾讯混元
- **服务层架构**: 模块化业务逻辑分离
- **前端框架**: HTML5 + CSS3 + JavaScript

## 安装和运行

### 1. 激活虚拟环境

```cmd
# 创建并激活虚拟环境
python -m venv venv
venv\Scripts\activate
```

### 2. 安装依赖

```cmd
cd f:\bishe\policy_qa_system
pip install -r requirements.txt
```

### 3. 设置 API Key

#### 3.1 智谱 API Key

获取智谱 API Key：
1. 访问 https://open.bigmodel.cn/
2. 注册/登录账号
3. 在控制台获取 API Key

#### 3.2 腾讯混元 API Key

获取腾讯混元 API Key：
1. 访问 https://cloud.tencent.com/product/hunyuan
2. 注册/登录账号
3. 在控制台获取 API Key

#### 3.3 设置环境变量

```cmd
# Windows
set SECRET_KEY=policy-qa-system-secret-key-2024
set ZHIPU_API_KEY=你的智谱API密钥
set TENCENT_API_KEY=你的腾讯API密钥

# Linux/Mac
export SECRET_KEY=policy-qa-system-secret-key-2024
export ZHIPU_API_KEY=你的智谱API密钥
export TENCENT_API_KEY=你的腾讯API密钥
```

### 4. 运行应用

```cmd
python run.py
```

### 5. 访问应用

- **首页**: http://localhost:5000/ - 政策问答和系统概览
- **登录**: http://localhost:5000/login - 用户登录/注册
- **管理后台**: http://localhost:5000/admin - 系统管理功能
- **讨论广场**: http://localhost:5000/forum - 用户讨论和民声
- **个人中心**: http://localhost:5000/profile - 用户个人信息和历史
- **搜索页面**: http://localhost:5000/search - 政策搜索功能

## 核心功能

### 1. 智能政策问答
- **RAG 技术**: 基于检索增强生成的智能问答
- **多轮对话**: 支持连续问答和上下文理解
- **政策解读**: 详细解释政策内容和适用条件
- **政策推荐**: 根据用户需求智能推荐相关政策
- **政策对比**: 对比分析不同政策的异同

### 2. 管理后台
- **政策管理**: 添加、编辑、删除政策，支持按分类和时间查询
- **分类管理**: 管理政策分类和标签
- **用户管理**: 角色管理、用户封禁和权限控制
- **专家管理**: 专家认证审核和管理
- **解读管理**: 专家解读内容审核和发布
- **民声管理**: 民声内容管理和推举机制
- **数据统计**: 系统数据概览和分析
- **RAG管理**: 构建和更新 RAG 数据库
- **反馈管理**: 用户反馈的处理和状态跟踪

### 3. 讨论广场
- **主题发布**: 用户发布讨论主题
- **回复互动**: 支持多级回复和讨论
- **点赞功能**: 每用户每条讨论只能点赞一次
- **实时更新**: 讨论内容实时刷新
- **民声推举**: 支持民声内容推举和展示

### 4. 个人中心
- **历史记录**: 查看搜索历史和问答历史
- **收藏管理**: 管理收藏的政策和解读
- **专家认证**: 申请和管理专家认证
- **个人信息**: 管理个人资料和昵称
- **历史言论**: 查看在讨论广场发布的内容

### 5. 安全保障
- **密码加密**: 使用 bcrypt 进行密码哈希
- **输入验证**: 防止恶意输入和 SQL 注入
- **权限控制**: 基于角色的访问控制
- **会话管理**: 安全的用户会话管理
- **用户反馈**: 用户反馈提交和处理机制

## 部署选项

### 1. 本地部署
- 适合开发和测试
- 直接运行 `python run.py`

### 2. Heroku 部署
- 适合生产环境
- 使用 `Procfile` 配置
- 支持自动部署和扩展

### 3. 腾讯云 Serverless
- 适合国内访问
- 使用 `serverless.yml` 配置
- 无服务器架构，按需计费

### 4. Vercel 部署
- 全球 CDN 加速
- 使用 `vercel.json` 配置
- 自动 HTTPS 支持

### 5. Docker 部署
- 容器化部署
- 使用 `Dockerfile` 配置
- 跨平台一致性

### 6. 阿里云轻量应用服务器
- 适合国内访问，速度快
- 配置 Ubuntu 20.04 + Nginx
- 适合毕设项目展示
